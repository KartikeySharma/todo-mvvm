# Todo List App

A simple todo list app written in Kotlin using Room Database.

## Features

- Add new todo items.
- Mark todo items as checked or unchecked.
- Delete todo items.
- View a list of all todo items.

## Getting Started

### Prerequisites

- Android Studio installed.
- Emulator or a physical Android device.
